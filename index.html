Коротко о моих проектах

Сборка проекта с помощью Gulp Используется шаблонизатор Twig JS
HTML, CSS, JS минимизированы Общие для всех страциц сайта скрипты скомпилированы в один файл
Изображения оптимизированы По желанию заказчика возможно предоставление всех исходных файлов
Используется препроцессор LESS Шрифты в формате base64
Цены обсуждаются индивидуально и зависимосит от сложности проекта

Оказываемые услуги

Дизайн

Разработка дизайна сайта, работа над стилистикой сайта, проработка графических и цветовых решений в зависимости от пожеланий клиента.

Верстка

Визуальное оформление сайта с использованием языка разметки HTML и CSS. Возможна работа по предосталяемым заказчиком макетам, но только при условии их надлежащего оформления.

Поддержка

Доработка сайтов после их запуска и обновление содержащейся в них информации (Только статичные сайты).

Мобильная адаптивность

Адаптация готового интренет-сайта под мобильные устройства.
